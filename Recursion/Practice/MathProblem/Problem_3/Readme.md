----------------------------------------------------------------------------------------

Given two numbers x and y find the product using recursion.

----------------------------------------------------------------------------------------

Examples : 

1. Input : x = 5, y = 2
Output : 10

2. Input : x = 100, y = 5
Output : 500

----------------------------------------------------------------------------------------

Method 
1) If x is less than y, swap the two variables value 
2) Recursively find y times the sum of x 
3) If any of them become zero, return 0
