---------------------------------------------------------------------------------------------------

Given a string calculate length of the string using recursion. 

---------------------------------------------------------------------------------------------------

Examples: 
1. Input : str = "abcd"
Output :4
2. Input : str = "GEEKSFORGEEKS"
Output :13

---------------------------------------------------------------------------------------------------

Algorithm:
recLen(str)
{
   If str is NULL 
       return 0
   Else 
      return 1 + recLen(str + 1)
}
