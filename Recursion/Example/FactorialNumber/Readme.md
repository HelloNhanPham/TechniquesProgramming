-------------------------------------------------------------------------------------

Write a program and recurrence relation to find the Factorial of n where n>2 .  

-------------------------------------------------------------------------------------

Mathematical Equation:
1 if n == 0 or n == 1;      
f(n) = n*f(n-1) if n> 1;

-------------------------------------------------------------------------------------

Recurrence Relation: 
T(n) = 1 for n = 0
T(n) = 1 + T(n-1) for n > 0

-------------------------------------------------------------------------------------

Recursive Program: 
Input: n = 5 
Output: 
factorial of 5 is: 120
