Tính tổng chữ số chẵn (lẻ) của N

---------------------------------------------------------------------------------------------------

IDEA:
     + Bài toán cơ sở : S(N) = 0 nếu N lẻ, N nếu N chẵn với N < 10
     + Công thức truy hồi : S(N) = S(N / 10) nếu N lẻ, N % 10 + S(N / 10) nếu N chẵn với N ≥ 10
