-----------------------------------------------------------------------------------

Given a sorted array and a number x, find a pair in an array whose sum is closest to x.

-----------------------------------------------------------------------------------

Examples:
1. Input: arr[] = {10, 22, 28, 29, 30, 40}, x = 54
Output: 22 and 30
2. Input: arr[] = {1, 3, 4, 7, 10}, x = 15
Output: 4 and 10
